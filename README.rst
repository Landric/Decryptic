=========
Decryptic
=========
-----------------------------
Decryptic Cryptanalysis Suite
-----------------------------

Decryptic is a simple suite of cryptanalysis tools, designed for analysing and decrypting unknown ciphertexts to reveal their true meaning.
Decryptic is made available under the GNU GPL and is copyright of Tom Blount. For more information see the LICENCE file.

Features

* Load ciphertexts from a file
* Display frequency analysis graph
